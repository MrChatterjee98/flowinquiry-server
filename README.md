Run the command `pnpm dev` that launch the Flexwork site in watch mode

```bash
> next -p 4000

  ▲ Next.js 14.2.14

 ✓ Starting...
   automatically enabled Fast Refresh for 4 custom loaders
 ✓ Ready in 930ms
```
